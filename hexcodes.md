Below are the common hex color codes formatted with the 0x prefix, which you can use directly with the .setColor command in Discord.js v14.

Common Hex Codes for Colors

Default Colors:

Default: 0x5865F2

# Basic Colors:

Black: 0x000000
White: 0xFFFFFF
Red: 0xFF0000
Lime: 0x00FF00
Blue: 0x0000FF
Yellow: 0xFFFF00
Cyan: 0x00FFFF
Magenta: 0xFF00FF

# Extended Colors:

Silver: 0xC0C0C0
Gray: 0x808080
Maroon: 0x800000
Olive: 0x808000
Green: 0x008000
Purple: 0x800080
Teal: 0x008080
Navy: 0x000080
